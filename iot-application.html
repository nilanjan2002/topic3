IoT Apps

    What is an IoT Application

  • IoT applications work with IoT devices to perform variety of tasks like edge computing, surveillance, generate real time insights from data collected.
  • The application layer sits above Perception layer and Network layer.

    Main Features

  • Secured Communication
  • Cross Platform
  • Seamless Communcation
  • Interactive and User-friendly UI

    Examples

  • Smart Home
  • Health Monitoring Wearables
  • Traffic Management
  • Environment Monitoring
  • Agriculture Management

Smart Building


    What does SMART mean?

  • In context of smart devices it means devices capable of making decision on its own based on algorithms.

    Main Features

  • Centralised monitoring and control system
  • Security
  • User friendly UX system
  • Sensors

    What are some Smart Building scenarios?

  • Smart Curtains
  • Automated lights
  • Biometric Security Systems
  • Temperature moderation for better sleep

Control Interface

    What are high-level features?

  • Door won’t unlock unless credentials match.
  • Curtains will open or close depending on light intensity.
  • Automated home cleaning.

    What are the needed parts?

  • Fingerprint sensors and camera for biometric security
  • DH22 Sensors
  • LDR sensors for smart curtains.
  • Infrared and photocell sensors for vacuum cleaners.
